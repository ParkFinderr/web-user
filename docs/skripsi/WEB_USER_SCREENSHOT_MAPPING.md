## Rekomendasi Format File Gambar
*   **Format**: PNG (`.png`) beresolusi tinggi untuk menjamin kualitas cetak dokumen skripsi yang tajam.
*   **Lokasi Penyimpanan Sementara**: Simpan seluruh file di atas ke dalam folder `docs/skripsi/screenshots/` (atau direktori lokal yang ditentukan peneliti) sebelum diimpor ke aplikasi pengolah dokumen (Microsoft Word / LaTeX).
*   **Metode Crop**: Dianjurkan mengambil gambar menggunakan rasio aspek mobile (jika disimulasikan via browser developer tools mode responsif mobile) karena aplikasi Web User ini dirancang responsif dan ramah pengguna mobile.
